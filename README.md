# Кэш объектов - Анализ стратегий вытеснения
## Описание проекта

Приложение демонстрирует работу кэша объектов с различными стратегиями вытеснения (FIFO, LRU, RAND). Реализовано кэширование слов при последовательном чтении текстовых файлов с возможностью анализа эффективности различных стратегий.

## Функциональность

- **Три стратегии вытеснения:**
  - **FIFO** (First In First Out) - вытесняется самый старый элемент
  - **LRU** (Least Recently Used) - вытесняется наименее используемый элемент
  - **RAND** (Random) - вытесняется случайный элемент

- **Возможности:**
  - Загрузка текстовых файлов
  - Настройка размера кэша (1-10000)
  - Выбор стратегии вытеснения
  - Сбор статистики работы кэша
  - Визуальное отображение результатов

## Технологии

- Java 17
- JavaFX 21
- Maven
- Паттерн "Стратегия" для реализации алгоритмов вытеснения

### Требования

- JDK 17 или выше
- Maven 3.6+

## Статистика

Приложение собирает следующую статистику:
- Всего слов в файле
- Уникальных слов
- Попаданий в кэш
- Промахов кэша
- Процент попаданий
- Количество вытеснений
- Заполненность кэша
